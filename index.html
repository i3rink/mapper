How to build your target area
1
Search a place in the header — e.g. Irvine. Pick a radius and hit Add point. You can add multiple centres.
2
Adjust the radius for any point using the dropdown in the sidebar list below the tools.
3
Refine the shape — use Draw area or Bounding box to add extra coverage beyond the circles.
4
Cut out areas you don't want — draw an Exclude zone over e.g. central Glasgow.
5
Generate targets — downloads an XLSX and map image ready for Google Ads.
